rutracker.py
============

Command-line tool that downloads torrent files from Rutracker.org.

Usage
-----

* `python rutracker.py -t <topic_id> -u <username> -p <password>`
* `python rutracker.py -u <username> -p <password> -i <file_with_topict_ids>`
